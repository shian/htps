# 河堤國小家庭聯絡簿 Web APP

![](https://travis-ci.org/shian/htps.svg?branch=master)

## 功能

* 學校公佈欄
* 班級聯絡簿

## 線上展示

http://shian.github.com/htps

## 開發

### 取得程式碼

```sh
git clone https://github.com/shian/htps.git
```
### 安裝相依套件

```sh
npm install -g bower polymer-cli && npm install && bower install
```

### 啟動開發用伺服器

```sh
polymer serve --compile never
```
